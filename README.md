# weather-app

Приложение погоды, используется API Open-Meteo [Weather Forecast](https://open-meteo.com/en/docs) 
и [Geocoding](https://open-meteo.com/en/docs/geocoding-api).

## Функции

1. Отображение текущей погоды в различных точках через виджеты
2. Поиск местоположений по названиям
3. Интеграция с картой Leaflet для выбора местоположения
4. Красивый интерфейс на Tailwind с иконками Lucide
5. Сохранение виджетов в localStorage
6. Автоматическое определение местоположения устройства

## Деплой

Приложение запущено на https://weather.komuk.ru/ через сервис Coolify. 
Сборка и конфигурация осуществляется автоматически через Nixpacks.
